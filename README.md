# Snake-Game-Python-
Simple Snake Game Using Python

This is a basic Snake game built using Python and the pygame library.
The project was created to understand how simple games work using loops,
conditions, and event handling.

---

## About the project

The player controls a snake using the arrow keys.
Each time the snake eats food, it grows longer.
The game ends if the snake hits the wall or itself.

This project focuses more on logic than visuals.

---

## How to run

1. Make sure Python is installed
2. Install pygame:
```bash
pip install pygame

Run the game:
python snake_game.py


Controls

Arrow keys → Move the snake

Close window → Exit game
